## 1. Аудит 13 применённых оптимизаций

| # | Оптимизация | Безопасна? | Проверка | Риски | Вердикт |
|---|------------|-----------|----------|-------|---------|
| 1 | **x86-64-v2 + Native CPU** | ✅ Да | `ld-linux` показывает x86-64-v2 supported. Sandy Bridge → SSE4.2, POPCNT | Нет | **Оставить** |
| 2 | **NR_CPUS 8192→8** | ✅ Да | `lscpu`: 8 CPU (4C/8T). Single socket | Нет | **Оставить** |
| 3 | **MAXSMP off** | ✅ Да | 1 NUMA node. Система не сервер | Нет | **Оставить** |
| 4 | **PCIe ASPM PERFORMANCE** | ✅ Да | Десктоп. RX 580 + SSD Btrfs. Энергопотребление не критично | +2-3W | **Оставить** |
| 5 | **DWARF ZSTD** | ✅ Да | Просто сжатие отладочной информации. BTF сохранён | Нет | **Оставить** |
| 6 | **EFI_MIXED off** | ✅ Да | `efibootmgr`: только 64-bit UEFI (Limine_x64.efi, BOOTX64.EFI) | Нет | **Оставить** |
| 7 | **SHA256** | ✅ Да | `mokutil`: Secure Boot не настроен | Нет | **Оставить** |
| 8 | **LSMs очищено** | ✅ Да | Отключены: SMACK, TOMOYO, LOADPIN, SafeSetID, IPE. Оставлены: AppArmor, SELinux, Yama, Landlock, Lockdown, Integrity, BPF | Нет | **Оставить** |
| 9 | **NUMA off** | ✅ Да | `lscpu`: 1 NUMA node. Одно-сокетная система | Нет | **Оставить** |
| 10 | **PREEMPT_LAZY** | ⚠️ Частично | Стабилен в ядре 7.x. CachyOS использует PREEMPT по умолчанию | Возможны микро-заикания в играх | **Оставить** (тестировать) |
| 11 | **THP MADVISE** | ✅ Да | 16GB RAM. MADVISE стабильнее для десктопа | Возможен минус FPS в некоторых играх | **Оставить** |
| 12 | **NUMA_BALANCING off** | ✅ Да | NUMA отключён, балансинг бессмыслен | Нет | **Оставить** |
| 13 | **SCHED_CLASS_EXT off** | ✅ Да | `ls /sys/fs/bpf/sched_ext`: не активен. CachyOS не использует sched_ext по умолчанию | Нет (можно включить при необходимости) | **Оставить** |

## 2. Аудит предложенных дополнительных оптимизаций

### 2.1 LTO/PGO/BOLT
| Оптимизация | Статус в CachyOS | Вердикт |
|------------|-----------------|---------|
| ThinLTO | ✅ **Уже включён** (`CONFIG_LTO_CLANG_THIN=y`) | Не трогать |
| FullLTO | ❌ Медленнее ThinLTO на 8 потоках | Не применять |
| PGO | ❌ Требует 2-этапного билда. Экспериментально | Не применять |
| BOLT | ❌ Нет поддержки в CachyOS | Не применять |

### 2.2 KVM off
- KVM модули: **не загружены** (`lsmod | grep kvm` → нет вывода)
- Docker/Podman: не проверено, но KVM может быть нужен для GPU passthrough
- **Вердикт:** KVM уже как модуль (`=m`). Менять на `=n` — избыточно, оставить как есть.

### 2.3 Legacy networks (FireWire, TokenRing, FDDI)
- FireWire: **не загружен** (`lsmod | grep firewire` → нет вывода)
- **Вердикт:** Безопасно отключить. Но в CachyOS config уже могут быть как модули. Эффект минимальный.

### 2.4 Old filesystems (FAT, NTFS, etc.)
| FS | Статус | Вердикт |
|----|--------|---------|
| **FAT/VFAT** | `/boot` = **FAT32** (`/dev/sda1 vfat`) | ❌ **КРИТИЧНО НЕ ОТКЛЮЧАТЬ** |
| NTFS | Нет NTFS разделов. `/mnt/games` = btrfs | ⚠️ Можно отключить, но нужен для внешних дисков |
| MSDOS | Не используется | ✅ Безопасно отключить |

### 2.5 Debug off
| Компонент | Статус | Вердикт |
|-----------|--------|---------|
| debugfs | **Смонтирован** (`/sys/kernel/debug`) | ❌ Не отключать |
| DEBUG_INFO | Уже сжат ZSTD | ✅ Текущее решение оптимально |
| DEBUG_KERNEL | Может использоваться для диагностики | ⚠️ Не трогать |

### 2.6 Ananicy-cpp
- `ananicy-cpp 1.2.0-1` — **установлен**
- `cachyos-ananicy-rules 1:1.1.40-1` — **установлен**
- **Вердикт:** ✅ Уже есть. Ничего не делать.

### 2.7 IRQ Balance
- `systemctl status irqbalance` → не установлен
- CachyOS использует свой механизм управления IRQ
- **Вердикт:** Не применять. CachyOS уже настроен.

### 2.8 Sysctl оптимизации
| Параметр | Текущее значение | Оценка |
|----------|-----------------|--------|
| `vm.swappiness` | 150 (CachyOS) | ✅ CachyOS настроил под ZRAM |
| `vm.dirty_ratio` | 0 (kernel-managed) | ✅ CachyOS |
| `net.core.default_qdisc` | fq_codel | ✅ Хорошо для десктопа |
| `net.ipv4.tcp_congestion_control` | **cubic** (не BBR!) | ⚠️ Можно предложить BBR3 |
| `fs.btrfs.metadata_ratio` | **НЕ СУЩЕСТВУЕТ** | ❌ Такого sysctl параметра нет |

**Вердикт:** Не менять sysctl — CachyOS уже настроил всё оптимально.

### 2.9 ZRAM
- **Уже настроен:** 15.6G, zstd, включён как swap
- **Вердикт:** Не трогать. CachyOS настроил.

### 2.10 EarlyOOM / Preload
| Пакет | Статус | Вердикт |
|-------|--------|---------|
| earlyoom | ❌ Не установлен | Можно добавить в optdepends |
| preload | ❌ Не установлен | Не рекомендуется (конфликтует с btrfs readahead) |

**Вердикт:** Добавить `earlyoom` в `optdepends` PKGBUILD.
